# SDDS Bunch Distribution Viewer

A GUI application for visualizing SDDS (Self Describing Data Sets) binary particle phase-space files generated by the [Elegant](https://www.aps.anl.gov/Accelerator-Operations-Physics/Software) particle accelerator simulation code.

Two versions are available:

| Version | File | GUI Framework | Status |
|---------|------|---------------|--------|
| v1 | `sddsbunchplot_v1.py` | CustomTkinter + Matplotlib | Stable |
| v2 | `SDDSbunchplot_v2.py` | PySide6 + Matplotlib | In development |

---

## SDDS File Format

The viewer reads binary SDDS files (`.sdds`) as produced by Elegant. Each file contains multiple **pages** (one per simulation step or pass), and each page contains:

- **8 columns per particle:** `x`, `xp`, `y`, `yp`, `t`, `p`, `dt`, `particleID`
- **Parameters per page:** `pCentral` (βγ), `PassCentralTime`, `Step`, `s`, and others

The parser handles both fixed-value header parameters (not stored in binary) and per-page binary parameters automatically.

## Features

### Both Versions

#### Multi-file Loading
- Load multiple SDDS files simultaneously
- Each file is assigned a distinct color
- Clickable file legend in the toolbar to select the active file for stats

#### Multi-Panel Grid
- Add and remove plot panels dynamically
- 1 panel → full width; 2–4 panels → 2 columns; 5+ panels → 3 columns
- Each panel independently configured with its own file, X axis, Y axis, and axis mode

#### Per-Panel Controls
- **File selector** — choose which loaded file to display
- **X / Y axis dropdowns** — any of the 8 data columns
- **Axis mode:**
  - `Auto` — plain matplotlib autoscale
  - `Roll` — smoothed window size anchored to current centroid
  - `Track` — mean ± N·σ, always centered on current frame
  - `Roll+Δ` / `Track+Δ` — same as above with reference subtraction (subtracts `pCentral` from `p`, `PassCentralTime` from `t`)
- **RF indicator** — glows green when RF bucket is active for that panel
- **Bkt toggle** — expands axes to show the full RF bucket

#### Plot Modes
- **Scatter** — individual particle dots with adjustable size and alpha
- **Heatmap 2D** — 2D histogram with Gaussian smoothing, multiple colormaps, linear/log color scale, colorbar

#### Stats Overlay (toggle)
- Crosshairs at distribution centroid (mean_x, mean_y)
- 1σ RMS ellipse from covariance matrix (solid, file color)
- 95% containment ellipse (dashed, 45% alpha)
- **Twiss parameter box** (top left) — emittance, beta, alpha, gamma — conjugate pairs only (x/xp, y/yp)
- **σ text box** (top right) — RMS width per axis; shows σ_δ = σ_p / p₀ for the momentum axis

#### Animation Playback
- Play/Pause button
- Frame slider for manual page navigation
- Adjustable speed (1–30 fps)

#### Keyboard Shortcuts (v1)
| Key | Action |
|-----|--------|
| `Space` | Play / Pause |
| `→` | Step forward one page |
| `←` | Step backward one page |
| `Ctrl+O` | Open file |

#### Particle Tracking
- Enter comma-separated particle IDs
- Trajectory line drawn across all pages
- Current position highlighted with a dot
- Up to 8 particles tracked simultaneously with distinct colors

#### Beam Loss
- Scans all pages for particles that disappear before the last page
- Red × markers shown at the last known position of lost particles

#### RF Bucket Separatrix
- Configurable via the **RF Bucket** dialog:
  - Particle species (Electron / Proton / Custom mass)
  - Momentum compaction factor αc
  - Revolution frequency f_rev (MHz)
  - Cavity table: voltage (V), harmonic number h, synchronous phase φs (degrees)
  - Multi-cavity support
  - **Ramp mode:** load a CSV with columns `Time, V1, h1, phi1, [V2, h2, phi2, ...]`
- Separatrix drawn as dashed yellow line, shifted to beam centroid
- Uses standard longitudinal Hamiltonian (above transition)

#### Correlation Matrix
- 6×6 scatter/histogram matrix for columns x, xp, y, yp, t, p
- Pearson correlation coefficient shown per off-diagonal panel

#### Stats Over Time Panel
- Docked panel showing mean ± 1σ (shaded), min/max (dotted) for all 7 data columns plus εx and εy across all pages
- Vertical line marks current page
- Shows the selected file only (click legend to select)

#### Session Save / Load
- Saves to JSON: file paths, all panel configurations, all sidebar settings, RF parameters, current page
- Restores everything on load including re-reading all SDDS files from their original paths

#### Export
- Exports all panels to PNG and PDF

## RF Ramp CSV Format

For ramp mode, the CSV should have one row per time step:

```
Time(s),  V1(V),  h1,  phi1(deg),  [V2(V),  h2,  phi2(deg), ...]
0.000,    500000, 1,   0.0
0.001,    600000, 1,   0.5
...
```

- Delimiter can be comma, tab, or whitespace — auto-detected
- Lines starting with `#` are treated as comments
- Additional cavity columns are optional

## Known Issues

- **RF separatrix alignment during ramp** — the separatrix may drift away from the bunch during an energy ramp. The centroid-shift approach is implemented but not fully validated for the ramp case. Works correctly for stationary buckets.
- **v1 axis mode dropdown** — occasionally opens at the top-left corner of the screen on some Linux window managers (known CustomTkinter bug, intermittent).
